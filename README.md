
## Description:
  Ai agent that returns the current exchange rate of any currency to another with the equivalent market prize.

# Purpose:
  To build and integrate an ai agent on #Telex platform.

## Technologies used:
  1. mastra framework.
  2. Typescript.
  3.  Node.js

## Dependencies:
    @mastra/core
    @mastra/loggers
    crypto
    dotenv
    openai
    zod


## devDependencies:
    @types/node
    mastra
    ts-node-dev
    typescript

## Getting started:

 1. run this command on your terminal:
     git clone [https://github.com/benjamin2orie/ai-agent.git]
  2 run cd ai-agent
  3 run npm install

## starting the mastra dev server:
   run this command:
   npm run dev

## .env contents:
  OPEN_API_KEY: "Your api key"
